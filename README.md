## Features

<table>
<tr>
<td width="50%">

**Writing**
- Rich Markdown editing
- Slash commands — type `/`
- Bubble menu on text selection
- Code blocks with syntax highlighting

</td>
<td width="50%">

**Workflow**
- Manual save (`Cmd+S` / `Cmd+Shift+S`)
- Modified file indicators in sidebar
- Find & replace in file and across files
- Dark & light themes

</td>
</tr>
</table>
